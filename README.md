# EcoDrone (Simple Website)

 This is a very simple responsive website made with only HTML and CSS.  
It is for a fictional product called EcoDrone.

## Includes:
- Responsive layout (basic CSS Grid)
- Very simple Flexbox section
- A form with:
  - Text input
  - Email input
  - Checkboxes
  - Radio buttons
  - Dropdown
  - Textarea
  - Submit button
- Three CSS-only interactions:
  1. Hover color change box
  2. Button press effect
  3. Card hover (very light)

## How to run
Open `index.html` in any browser.
